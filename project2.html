Re-design of Absolute Collagen Web Shop

E-commerce, Mobile, Web

This project focused on improving the user experience of the Absolute Collagen web shop. I simplified the customer flow and unified the visual language system to make it consistent and recognizable.

Key improvements included enhancing the information hierarchy, optimizing the loading times, and addressing the ethical considerations of UX choices.
